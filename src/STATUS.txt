ПРОЕКТ: Сайт для записи на настольные игры.
ТЕКУЩИЙ СТАТУС:
Фронтенд на React + Vite + Tailwind v4 настроен и работает.
Реализована главная страница с плиткой игр (бело-зеленый дизайн).
Работает поиск по играм и имитация перехода на страницу игры через состояние (useState).
Все данные пока в заглушках (MOCK_GAMES).
СЛЕДУЮЩИЙ ШАГ:
Разработка Бэкенда на Python (FastAPI + PostgreSQL). Нужно создать API, чтобы заменить заглушки реальными данными и реализовать авторизацию (Google/Yandex).
